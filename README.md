# Telegram Mini App

এই প্রোজেক্টটি একটি ছোট লোকাল ওয়েব অ্যাপ, যেখানে আপনি পোস্ট করতে পারবেন, পোস্টগুলো পড়তে পারবেন, লাইক করতে পারবেন এবং মন্তব্য করতে পারবেন।

## চালানোর নিয়ম

1. `index.html` ফাইলটি ব্রাউজারে খুলুন
2. পোস্ট লেখুন এবং "পোস্ট করুন" বোতনে ক্লিক করুন
3. প্রতিটি পোস্টে লাইক এবং মন্তব্য যোগ করতে পারবেন

## ফাইল

- `index.html` - অ্যাপের প্রধান সেকশন
- `styles.css` - ডিজাইন ও লেআউট
- `app.js` - পোস্ট, লাইক, কমেন্ট লজিক
